# Image Overlay Tool (Browser Version)

Vercelで動作するように、完全にブラウザ上で動作するバージョンに書き換えたものです。

## 概要
JPEG画像URLのリスト（CSV: 1列目にURLのみ記述）とオーバーレイ画像（PNG）を読み込み、ブラウザ上で合成してZipファイルとしてダウンロードします。
サーバーでの画像処理を行わないため、Vercelのタイムアウト制限を受けずに大量の処理が可能です。
出力ファイル名は、画像URLの末尾（ファイル名部分）がそのまま使用されます。

## デプロイ方法 (Vercel)

1. このフォルダ全体をGitHubリポジトリにアップロードします。
2. Vercelのダッシュボードで "Add New..." -> "Project" を選択します。
3. アップロードしたGitHubリポジトリを選択します。
4. 設定はすべてデフォルトのまま "Deploy" をクリックします。

## 構成
- `index.html`: メイン画面
- `script.js`: 画像処理ロジック（クライアントサイド）
- `api/proxy.js`: 画像取得用プロキシ（CORS回避用）
- `package.json`: プロジェクト設定ファイル

## 注意点
- 画像のダウンロードには `api/proxy.js` を経由するため、デプロイ先のドメインからのアクセスとなります。
- ブラウザのメモリを使用するため、極端に巨大な画像や数万枚の処理を行うとブラウザが重くなる可能性があります（並列数は5に制限しています）。
